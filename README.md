# 🛰️ Drone Mission Frontend

This is the frontend interface for the Drone Mission System, built using Lovable (AI code assistant). It allows mission planning, drone monitoring, and real-time path simulation.

---
## Installation

 - npm install --force
---

---

## ✨ Features

- Interactive map-based polygon mission planner
- Drone list and live status view
- Real-time mission control: pause, resume, abort
- Display of flight paths and battery status

---

## ⚙️ Tools Used

- Lovable.ai
- HTML/CSS/JS (generated)
- REST API integration with Django backend
- Google maps for survey-area and flight-path  
---

## 📂 Pages

| Page             | Description                           |
|------------------|---------------------------------------|
| Dashboard        | Overview of all drones and missions   |
| Drone Details    | Shows drone status and battery        |
| Mission Planner  | Create polygons and generate missions |
| Mission Monitor  | Live mission status & control buttons |
